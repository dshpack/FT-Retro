# FT-Retro
Retro space shoter - you need to kill all enemies!!!!

#### Features:
+ Multiplayer
+ Score counter
+ Game timer
+ Enemy shoots on you and you may shot him
+ Enemy can ram you

#### 
"XD" - player;
"HO" - enemy;
"-" - player bullets;
